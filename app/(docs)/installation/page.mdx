# Installation

To use our component library in your project, follow these steps:

1. Install the package using npm or yarn:

```bash
npm install our-component-library
# or
yarn add our-component-library
```

2. Import the components you need in your React files:

```jsx
import { Button } from 'our-component-library'
```

3. Start using the components in your application!
